# TrailMap

Offline topographic map app for Android with GPS track recording, SRTM-based 3D terrain visualization, and elevation profiles. Built for hiking, mountain biking, and motorcycling.

## Features

- **Offline vector maps** (OpenAndroMaps) — download once, use anywhere with no signal
- **GPS track recording** with automatic elevation correction using SRTM terrain data
- **3D terrain viewer** — see your ride or hike rendered on real terrain, with the route overlaid
- **2D elevation profile** with interactive distance/elevation readout
- **Waypoints** — drop a pin, navigate back to it with bearing and distance
- **Offline POI search** (trails, campgrounds, etc.)
- **GPS satellite signal display**
- **Crash recovery** — interrupted recordings can be recovered and saved on next launch

## Installation

1. Download `TrailMap.apk` from the [latest release](../../releases/latest)
2. On your phone, enable **Install unknown apps** for your browser or file manager (Settings > Security)
3. Open the downloaded APK and tap **Install**

Full instructions, including map downloads and 3D viewer controls, are in the [User Manual](TrailMap_Manual.pdf).

## Requirements

- Android 8.0 (Oreo) or later
- GPS-capable device
- Internet connection for initial map and elevation downloads only — fully offline afterward

## Getting Started

1. Open the app and grant location permission
2. Open **Map Manager** (menu icon) and download a map for your region — this also downloads POI data and SRTM elevation
3. Activate the downloaded map
4. Tap **Record** to start a GPS track
5. After your ride, tap a track in the track list and select **View 3D** to see it on the terrain
